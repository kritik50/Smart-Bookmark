Elevator Pitch (60% AI Focus)

"I built SmartMark, an intelligent bookmarking platform that uses Retrieval-Augmented Generation (RAG) to eliminate digital clutter. When a user saves a link, a custom Node.js web scraper extracts the raw HTML, strips the noise, and feeds the content to Google's Gemini 2.0 model. The AI processes the text to generate concise summaries, extracts OpenGraph metadata, and outputs structured JSON to auto-tag the article. To solve the problem of losing old bookmarks, I implemented a Semantic Search engine using Gemini's text-embedding models and Supabase's `pgvector` extension. This converts the AI summaries into 768-dimensional vectors, allowing users to search their library by conceptual meaning and context rather than relying on exact keyword matches. The entire system is wrapped in a real-time Next.js and React dashboard, offering an enterprise-grade, lightning-fast UX."
